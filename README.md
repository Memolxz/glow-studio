## 🛠️ Comandos
Comandos basicos de git\
```
git add .
```
```
git commit -m "mensaje"
```
```
git push
```
Para instalar las dependencias:
```
pnpm install
```
Comandos de Prisma
```
pnpm prisma generate
```
```
pnpm prisma migrate dev --name nombre-de-migracion
```
```
pnpm prisma migrate status
```
